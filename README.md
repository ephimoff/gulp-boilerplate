# Gulp Boilerplate Template

A good example of Gulp tasks and how they can be used for creating frontend projects.
A first step to replace your static site generator.

## Features

- [Handelbars] template engine to render HTML
- Supports [Stylus] pre-processor to write CSS
- Optimises images
- [Auto-reloading]
- Support Markdown and generates pages out of `*.md`

[Handelbars]: http://handlebarsjs.com
[Stylus]: http://stylus-lang.com
[Auto-reloading]: https://www.browsersync.io
